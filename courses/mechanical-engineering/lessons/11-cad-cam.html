Introduction to CAD/CAM

Computer-Aided Design (CAD) and Computer-Aided Manufacturing (CAM) are essential tools for modern mechanical engineering. CAD enables precise 3D modeling and analysis, while CAM translates designs into machine instructions for automated manufacturing.

CAD Fundamentals

Modeling Approaches

2D Drafting: Traditional approach (computer-based) Lines, arcs, circles, dimensions Similar to manual drafting Outputs: Engineering drawings Legacy systems still in use Being replaced by 3D 3D Modeling: Three-dimensional virtual representation Advantages: + Visualize complete design + Detect interferences + Calculate mass properties + Generate drawings automatically + Simulation (FEA, CFD, motion) Types: Wireframe: Lines and curves (limited use) Surface: Sheet-like surfaces (complex shapes) Solid: Enclosed volume (most common) Modeling Philosophies: Parametric (Feature-Based): History-based design tree Dimensions drive geometry Features: Extrude, revolve, sweep, loft Holes, fillets, chamfers Patterns, mirrors Constraints: Geometric: Parallel, perpendicular, tangent Dimensional: Lengths, angles, radii Advantages: + Design intent captured + Easy to modify (change parameters) + Relationships maintained Disadvantages: - Feature failures if poorly structured - Learning curve Software: SolidWorks, Inventor, Creo, NX Direct (Explicit) Modeling: No history, edit geometry directly Push, pull, move faces Immediate results Advantages: + Fast for simple edits + Good for imported geometry + No feature failures Disadvantages: - Design intent not captured - Harder to make systematic changes Software: SpaceClaim, Onshape (some tools) Hybrid Approach: Combine parametric and direct Modern CAD trend Best of both worlds Top-Down vs Bottom-Up: Bottom-up: Design parts, then assemble Traditional approach Top-down: Design in assembly context Create parts from assembly Better for complex products

Sketching and Constraints

2D Sketch: Foundation of 3D features Drawn on plane or face Sketch Entities: Line, arc, circle, spline Rectangle, polygon Construction geometry (reference only) Constraints (Relations): Geometric Constraints: Horizontal/Vertical: Force orientation Parallel/Perpendicular: Between lines Tangent: Smooth connection Concentric: Share center Coincident: Points touch Collinear: On same line Equal: Same length/radius Symmetric: Mirror about centerline Dimensional Constraints: Length, angle, radius, diameter Numerical values drive size Can be: - Fixed: Constant value - Driven: Calculated from others - Linked: Tied to parameter/equation Sketch States: Under-defined (blue): Needs constraints Can move freely (unintentional) Fully-defined (black): Completely constrained Cannot move (ideal for manufacturing) Over-defined (red/orange): Conflicting constraints ERROR - must resolve Best Practices: 1. Start with construction lines (centerlines, axes) 2. Draw approximate geometry 3. Add geometric constraints first 4. Add dimensions last 5. Fully define critical features 6. Use design intent (what should stay fixed) Global Variables: Define once, use everywhere Example: Plate_Thickness = 5 mm Hole_Diameter = Plate_Thickness + 0.5 Change Plate_Thickness ‚Üí all update Equations: Relate dimensions mathematically Length_2 = Length_1 * 1.5 Angle_2 = 180¬∞ - Angle_1 Propagate changes automatically

3D Features and Modeling

Common Features

Base Features: Extrude: Extend 2D sketch perpendicular to plane Options: - Blind: Specified distance - Through all: Penetrate all - Up to surface/face - Mid-plane: Symmetric Can add or remove material Revolve: Rotate sketch around axis Creates cylindrical/conical parts Angle: 360¬∞ (full) or partial Sweep: Follow sketch along path Profile sketch + path sketch Applications: - Pipes, tubes, handles - Complex profiles Loft: Transition between multiple profiles Smooth blend (organic shapes) Guide curves for control Applications: - Bottles, housings - Aerodynamic surfaces Applied Features: Hole: Specialized feature (countersink, counterbore, tapped) Standards: ANSI, ISO Faster than sketch + extrude Fillet: Round internal/external edges Constant or variable radius Reduces stress concentration Manufacturing-friendly Chamfer: Beveled edge (45¬∞ common) Distance-distance or distance-angle Deburring, assembly clearance Shell: Hollow out solid Uniform wall thickness One or more faces removed (opening) Draft: Taper for mold release Angle from pull direction Essential for casting/molding Pattern: Replicate features Linear: Rows and columns Circular: Around axis Curve-driven: Along path Instances linked (change one ‚Üí all change) Mirror: Reflect features/parts about plane Ensures symmetry Half the work! Rib/Web: Thin structural support Strengthen without much weight

Assembly Modeling

Assembly: Multiple parts positioned together Relationships define relative positions Mates (Constraints): Coincident: Faces, edges, or points touch Planar faces: Coplanar Axes: Aligned Concentric: Cylindrical faces share axis Common: Shaft in hole Parallel/Perpendicular: Maintain angular relationship Distance: Specific gap between faces Angle: Specific angle between planes/lines Tangent: Curved faces touch Lock: Fix position rigidly Degrees of Freedom: 6 DOF: 3 translations (X, Y, Z) + 3 rotations Fully constrained: 0 DOF (fixed) Under-constrained: > 0 DOF (can move) Assembly Techniques: In-Context Design: Design part within assembly Reference other parts (holes align, etc.) Maintains relationships Risk: External references can break Skeleton Modeling: Master layout part/sketch Other parts reference skeleton Centralized control Interference Detection: Identify overlapping volumes Clearance analysis Prevent collisions Exploded View: Show separated components For assembly instructions, presentations Bill of Materials (BOM): List of all parts and quantities Exported to spreadsheet Manufacturing input Motion Simulation: Define joints (revolute, slider, etc.) Animate mechanism Check range of motion

Engineering Drawings

Drawing Standards and Views

Purpose: Communicate design for manufacturing 2D representation of 3D part Complete dimensional definition Standards: ASME Y14.5 (US) ISO 1101 (International) Ensure consistent interpretation Projection Methods: Third-Angle Projection (US): Object between observer and projection plane Front view: What you see from front Top view: Above front view Right view: Right of front view First-Angle Projection (ISO): Projection plane between observer and object Top view: Below front view Right view: Left of front view View Selection: Choose most descriptive as front view Minimum views needed (typically 3) Avoid hidden lines if possible Auxiliary views: Inclined surfaces Section Views: Cut through part to show interior Types: Full section: Cut completely through Half section: Quarter removed (symmetric) Offset section: Staggered cutting plane Broken-out: Local area only Section lines (hatching): Indicate cut surface Detail View: Enlarged portion Show small features clearly Dimensioning: Rules: - Dimension each feature once - Dimension to visible features - Place between views when possible - Avoid redundancy - Group related dimensions Dimension Types: Linear: Length, width, height Angular: Degrees Radial: Radius (R prefix) Diametral: Diameter (√ò symbol) Tolerances: Allowable variation Methods: Limit: Max and min (25.2/25.0) Plus-minus: Nominal ¬± tolerance (25.1 ¬± 0.1) Unilateral: One direction (25.0 +0.2/-0) Bilateral: Both directions (25.1 ¬± 0.1) General tolerance block: Default for uncalled Fits: Shaft-hole relationships Clearance: Hole > shaft (sliding) Interference: Shaft > hole (press-fit) Transition: Can be either ISO system: H7/g6, etc. Letter: Tolerance zone Number: Precision grade

Finite Element Analysis (FEA)

FEA Fundamentals

Purpose: Simulate real-world behavior Predict: - Stress and deformation - Vibration modes - Heat transfer - Fluid flow Before physical testing FEA Process: 1. Preprocessing: a. Geometry: Import/create CAD model Simplify (remove fillets, small holes if allowed) b. Material: Define properties E, ŒΩ, œÅ, thermal, etc. c. Mesh: Divide into elements Nodes and elements Element types: Solid: Tetrahedral (4-node), hex (8-node) Shell: For thin parts (2D elements) Beam: For long slender (1D elements) Mesh refinement: Coarse: Fast, less accurate Fine: Slow, more accurate Adaptive meshing: Auto-refine high-stress areas d. Boundary Conditions: Fixtures: Constrain DOF (prevent rigid body motion) Fixed: All DOF Roller: Some DOF free Loads: Forces, pressures, torques Point load, distributed load, pressure Contacts: Part interactions Bonded, no penetration, friction 2. Solution: Solve system of equations Ax = b Where A = stiffness matrix (large!) Iterative solvers Convergence criteria 3. Postprocessing: View results Contour plots: - von Mises stress - Displacement magnitude - Safety factor (yield/stress) Animations: Deformation, mode shapes Graphs: Stress vs location Reports: Max stress, displacement Result Interpretation: Stress: von Mises: Compare to yield strength Red zones: High stress (potential failure) Check: œÉ_max < œÉ_y/n (factor of safety n) Displacement: Maximum deflection Check against requirements Stiffness adequate? Factor of Safety: n = œÉ_y / œÉ_von_Mises Minimum value critical n < 1: Failure predicted! n > design requirement: Acceptable Singularities: Infinite stress at sharp corners, point loads Not real (local mesh refinement) Interpret nearby stress instead Mesh Convergence: Run with finer mesh If results don't change ‚Üí converged If changing ‚Üí refine more Validation: Compare to: - Hand calculations (simple cases) - Physical testing - Published data Build confidence in model

Computer-Aided Manufacturing (CAM)

CAM Fundamentals

CAM Purpose: Generate toolpaths for CNC machines Input: CAD model Output: G-code (machine instructions) CNC (Computer Numerical Control): Automated machine tool control Types: - Mills (3-axis, 4-axis, 5-axis) - Lathes (turning) - Routers - Laser cutters - 3D printers CAM Workflow: 1. Import Geometry: CAD model (STEP, IGES, native) Verify: Orientation, units, scale 2. Define Stock: Raw material block Size and shape (rectangular, cylindrical) 3. Setup: Work Coordinate System (WCS): Origin (X0, Y0, Z0) Usually: Corner or center of part Fixtures: How part held Vise, clamps, vacuum table 4. Select Operations: Machining strategies for features Order matters (rough ‚Üí finish) 5. Define Toolpaths: Tool motion to remove material 6. Simulate: Verify toolpaths (no collisions, gouges) Material removal animation 7. Post-Process: Convert to machine-specific G-code Post-processor for each machine 8. Output: G-code file to machine Machining Strategies: Facing: Flatten top surface Large tool, light depth 2D Pocket: Remove material inside closed boundary Zig-zag, offset spiral, etc. 2D Contour: Follow edges Finishing operation Drilling: Peck, deep hole cycles 3D Surface: Sculpted surfaces Waterline, spiral, radial Turning (Lathe): OD/ID turning, facing, grooving, threading Toolpath Parameters: Stepover: Distance between adjacent passes % of tool diameter Smaller ‚Üí better finish, longer time Stepdown: Depth per pass Axial depth of cut Roughing: Larger (remove material fast) Finishing: Smaller (good surface) Speeds and Feeds: Spindle speed (RPM) Feed rate (mm/min or in/min) From material, tool, operation Too fast: Tool wear, breakage Too slow: Inefficient, rubbing

Summary

  • CAD fundamentals: 3D parametric modeling most common, sketches with constraints drive features (extrude, revolve)
  • Constraints: geometric (parallel, tangent) + dimensional (lengths, angles) to fully define sketches
  • Features: extrude/revolve (base), fillets/chamfers (applied), patterns/mirrors (replicate), assemblies with mates
  • Engineering drawings: orthographic views, section views, dimensioning with tolerances, GD&T for precision control
  • GD&T: position ‚äï, perpendicularity ‚ä•, flatness ‚è•, etc. with datums (A, B, C) and material condition ‚ìÇ MMC
  • FEA workflow: mesh geometry ‚Üí apply loads/fixtures ‚Üí solve ‚Üí analyze stress/displacement/safety factor
  • FEA types: static (stress), modal (vibration), thermal (heat), nonlinear (plasticity), topology optimization (material distribution)
  • CAM: define stock ‚Üí create toolpaths (pocket, contour, drill) ‚Üí simulate ‚Üí post-process to G-code
  • G-code: G00 rapid, G01 linear cut, G02/G03 arcs, M03 spindle on, coordinates absolute G90 or incremental G91
  • Modern design: CAD for geometry, FEA for validation, CAM for manufacturing, integrated PLM systems manage lifecycle
