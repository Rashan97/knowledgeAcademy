Lesson 6: C++ Basics

üìñ 100 min read üìä Intermediate

Welcome to C++

C++ is a powerful, high-performance programming language used for system software, game engines, embedded systems, and applications where speed is critical. It extends C with object-oriented features while maintaining low-level control over hardware.

üìå Why Learn C++?

C++ powers major systems: Windows, macOS, Adobe applications, game engines (Unreal, Unity), browsers (Chrome, Firefox), databases (MySQL, MongoDB), and much more. It's the backbone of performance-critical software.

C++ Syntax and Compilation

Basic Program Structure

#include <iostream> // Preprocessor directive // Main function - program entry point int main() { std::cout << "Hello, World!" << std::endl; return 0; // Return 0 for success } /* Compile and run: g++ hello.cpp -o hello ./hello */

Using Namespace std

#include <iostream> using namespace std; // Avoid typing std:: int main() { cout << "No std:: prefix needed" << endl; return 0; }

‚ö†Ô∏è Namespace Best Practice

In small programs, using namespace std; is fine. In larger projects or header files, avoid it to prevent naming conflicts. Use std:: explicitly.

Data Types and Variables

Fundamental Types

#include <iostream> using namespace std; int main() { // Integer types int age = 25; // Usually 4 bytes short smallNum = 100; // 2 bytes long bigNum = 1000000; // 4 or 8 bytes long long hugeNum = 9999999999; // 8 bytes // Floating-point types float price = 19.99f; // 4 bytes, 'f' suffix double pi = 3.14159265359; // 8 bytes, more precise // Character and boolean char grade = 'A'; // Single quotes! bool isStudent = true; // true or false // String (from standard library) string name = "Alice"; // Requires #include <string> // Auto type deduction (C++11) auto x = 42; // int auto y = 3.14; // double auto z = "text"; // const char* // Print sizes cout << "Size of int: " << sizeof(int) << " bytes" << endl; return 0; }

Pointers and References

Pointers and references are C++'s most powerful (and dangerous) features. They provide direct memory access and enable efficient data manipulation.

Pointers

int main() { int x = 10; int* ptr = &x; // ptr stores address of x cout << "Value of x: " << x << endl; // 10 cout << "Address of x: " << &x << endl; // Memory address cout << "Value of ptr: " << ptr << endl; // Same address cout << "Value at ptr: " << *ptr << endl; // 10 (dereferencing) // Modify through pointer *ptr = 20; cout << "New value of x: " << x << endl; // 20 // Null pointer int* nullPtr = nullptr; // C++11 way (safer than NULL) return 0; }

References

int main() { int x = 10; int& ref = x; // ref is an alias for x cout << "x: " << x << ", ref: " << ref << endl; // Both 10 ref = 20; // Modifying ref changes x cout << "x: " << x << endl; // 20 // References must be initialized and cannot be reassigned int y = 30; // ref = y; // This changes x to 30, doesn't rebind ref to y! return 0; }

üí° Pointers vs References

  • References: Safer, must be initialized, cannot be null, cannot be reassigned
  • Pointers: More flexible, can be null, can be reassigned, allow pointer arithmetic
  • Use references for function parameters when possible

Functions

Function Declaration and Definition

#include <iostream> using namespace std; // Function declaration (prototype) int add(int a, int b); void printMessage(string msg); int main() { int result = add(5, 3); cout << "Result: " << result << endl; printMessage("Hello from function!"); return 0; } // Function definition int add(int a, int b) { return a + b; } void printMessage(string msg) { cout << msg << endl; }

Pass by Value, Reference, and Pointer

// Pass by value (copy) void incrementValue(int x) { x++; // Only modifies local copy } // Pass by reference (no copy, can modify original) void incrementRef(int& x) { x++; // Modifies original } // Pass by const reference (efficient, read-only) void printVector(const vector<int>& vec) { for (int num : vec) { cout << num << " "; } } // Pass by pointer void incrementPointer(int* x) { (*x)++; // Dereference and increment } int main() { int num = 10; incrementValue(num); cout << num << endl; // 10 (unchanged) incrementRef(num); cout << num << endl; // 11 (changed) incrementPointer(&num); cout << num << endl; // 12 (changed) return 0; }

Function Overloading

// Same function name, different parameters int add(int a, int b) { return a + b; } double add(double a, double b) { return a + b; } int add(int a, int b, int c) { return a + b + c; } int main() { cout << add(5, 3) << endl; // Calls int version cout << add(5.5, 3.2) << endl; // Calls double version cout << add(1, 2, 3) << endl; // Calls 3-parameter version return 0; }

Arrays and Strings

C-Style Arrays

int main() { // Static array int numbers[5] = {1, 2, 3, 4, 5}; // Access elements cout << numbers[0] << endl; // 1 numbers[2] = 30; // Iterate for (int i = 0; i < 5; i++) { cout << numbers[i] << " "; } cout << endl; // Range-based for loop (C++11) for (int num : numbers) { cout << num << " "; } return 0; }

C++ Strings

#include <string> int main() { string name = "Alice"; string greeting = "Hello, " + name; // Concatenation // String methods cout << greeting.length() << endl; // 12 cout << greeting.substr(0, 5) << endl; // "Hello" cout << greeting.find("Alice") << endl; // 7 // Character access cout << name[0] << endl; // 'A' // Comparison string str1 = "apple"; string str2 = "banana"; if (str1 < str2) { cout << str1 << " comes before " << str2 << endl; } return 0; }

Memory Management

Dynamic Memory Allocation

int main() { // Allocate single variable int* ptr = new int; *ptr = 42; cout << *ptr << endl; delete ptr; // Free memory // Allocate array int* arr = new int[5]; for (int i = 0; i < 5; i++) { arr[i] = i * 10; } delete[] arr; // Free array memory // ALWAYS delete what you new! return 0; }

‚ö†Ô∏è Memory Management Rules

  • Every new must have a corresponding delete
  • Array allocation (new[]) requires array deletion (delete[])
  • Memory leaks occur when you forget to delete
  • Modern C++ prefers smart pointers (covered in next lesson)

STL Containers

The Standard Template Library (STL) provides powerful, tested data structures.

Vector (Dynamic Array)

#include <vector> int main() { vector<int> numbers; // Add elements numbers.push_back(10); numbers.push_back(20); numbers.push_back(30); // Access elements cout << numbers[0] << endl; // 10 cout << numbers.at(1) << endl; // 20 (with bounds checking) // Size and capacity cout << "Size: " << numbers.size() << endl; // Iterate for (int num : numbers) { cout << num << " "; } // Initialize with values vector<int> scores = {85, 90, 78, 92}; return 0; }

Map (Key-Value Pairs)

#include <map> int main() { map<string, int> ages; // Insert elements ages["Alice"] = 25; ages["Bob"] = 30; ages.insert({"Charlie", 35}); // Access elements cout << ages["Alice"] << endl; // 25 // Check existence if (ages.find("Bob") != ages.end()) { cout << "Bob found!" << endl; } // Iterate for (const auto& pair : ages) { cout << pair.first << ": " << pair.second << endl; } return 0; }

Set (Unique Values)

#include <set> int main() { set<int> numbers; // Insert (duplicates ignored) numbers.insert(5); numbers.insert(2); numbers.insert(8); numbers.insert(2); // Ignored // Automatically sorted: 2, 5, 8 for (int num : numbers) { cout << num << " "; } // Check existence if (numbers.count(5)) { cout << "5 exists" << endl; } return 0; }

Input/Output Streams

Console I/O

int main() { string name; int age; // Input cout << "Enter name: "; cin >> name; // Stops at whitespace cout << "Enter age: "; cin >> age; // Get full line cin.ignore(); // Clear newline from buffer cout << "Enter full address: "; string address; getline(cin, address); // Output cout << "Hello, " << name << "! You are " << age << " years old." << endl; return 0; }

File I/O

#include <fstream> int main() { // Write to file ofstream outFile("data.txt"); if (outFile.is_open()) { outFile << "Hello, File!" << endl; outFile << "Line 2" << endl; outFile.close(); } // Read from file ifstream inFile("data.txt"); string line; if (inFile.is_open()) { while (getline(inFile, line)) { cout << line << endl; } inFile.close(); } return 0; }

Problem 2: Grade Calculator

Calculate average grade from a vector of scores.

#include <vector> #include <numeric> double calculateAverage(const vector<int>& grades) { if (grades.empty()) return 0.0; int sum = accumulate(grades.begin(), grades.end(), 0); return static_cast<double>(sum) / grades.size(); } int main() { vector<int> grades = {85, 90, 78, 92, 88}; cout << "Average: " << calculateAverage(grades) << endl; return 0; }

Key Takeaways

  • C++ requires compilation before running
  • Static typing with explicit type declarations
  • Pointers provide memory addresses; references are safer aliases
  • Function overloading allows same name with different parameters
  • Manual memory management with new/delete (use smart pointers in modern C++)
  • STL containers (vector, map, set) provide robust data structures
  • Pass by reference for efficiency with large objects
  • Always match new with delete, new[] with delete[]
