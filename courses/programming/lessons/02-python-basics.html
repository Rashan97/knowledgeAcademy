Lesson 2: Python Basics

üìñ 90 min read üìä Beginner

Why Python?

Python is one of the most popular programming languages in the world, and for good reason. It's easy to read, has a gentle learning curve, and is incredibly powerful. Created by Guido van Rossum in 1991, Python emphasizes code readability and simplicity.

üìå Python Philosophy

"There should be one-- and preferably only one --obvious way to do it."
- The Zen of Python (type import this in Python!)

Your First Python Program

Let's start with the traditional "Hello, World!" program:

# This is a comment - Python ignores these print("Hello, World!") # Output: Hello, World!

That's it! Unlike many languages, Python doesn't require semicolons, complex syntax, or boilerplate code. Just write what you mean.

Variables and Data Types

Creating Variables

In Python, you don't need to declare variable types - Python figures it out automatically:

# Numbers age = 25 price = 19.99 temperature = -5.5 # Strings (text) name = "Alice" city = 'New York' # Single or double quotes work # Booleans (True/False) is_student = True is_employed = False # Print variables print(name) # Output: Alice print(age) # Output: 25 print(is_student) # Output: True

Variable Naming Rules

  • Must start with a letter or underscore (_)
  • Can contain letters, numbers, and underscores
  • Case-sensitive (name and Name are different)
  • Cannot use Python keywords (if, for, while, etc.)
  • Use descriptive names: user_age not ua
# Good variable names student_name = "Bob" total_price = 99.99 is_valid = True MAX_SIZE = 100 # Constants in CAPS # Bad variable names # 1name = "Invalid" # Starts with number # my-var = 5 # Contains hyphen # class = "test" # Reserved keyword

Basic Data Types

1. Numbers

# Integers (whole numbers) count = 10 negative = -5 large_number = 1_000_000 # Underscores for readability # Floats (decimal numbers) pi = 3.14159 percentage = 0.75 # Arithmetic operations sum_result = 10 + 5 # 15 difference = 10 - 5 # 5 product = 10 * 5 # 50 quotient = 10 / 5 # 2.0 (always returns float) floor_division = 10 // 3 # 3 (integer division) remainder = 10 % 3 # 1 (modulo) power = 2 ** 3 # 8 (exponentiation) print(f"Sum: {sum_result}") print(f"Power: {power}")

2. Strings

# Creating strings message = "Hello, Python!" multiline = """This is a multi-line string""" # String operations first_name = "John" last_name = "Doe" full_name = first_name + " " + last_name # Concatenation # String methods text = "python programming" print(text.upper()) # PYTHON PROGRAMMING print(text.capitalize()) # Python programming print(text.title()) # Python Programming print(text.replace("python", "Python")) print(text.split()) # ['python', 'programming'] # String indexing and slicing word = "Python" print(word[0]) # P (first character) print(word[-1]) # n (last character) print(word[0:3]) # Pyt (slice from 0 to 3) print(word[2:]) # thon (from 2 to end) print(word[:4]) # Pyth (from start to 4) # String formatting (f-strings - Python 3.6+) name = "Alice" age = 30 print(f"My name is {name} and I am {age} years old") print(f"Next year I'll be {age + 1}")

üí° String Tips

  • Strings are immutable - you can't change them, only create new ones
  • Use f-strings for formatting (fastest and most readable)
  • Triple quotes (""" or ''') for multi-line strings

3. Lists (Arrays)

# Creating lists numbers = [1, 2, 3, 4, 5] fruits = ["apple", "banana", "cherry"] mixed = [1, "hello", 3.14, True] # Can mix types! empty = [] # Accessing elements print(fruits[0]) # apple (first item) print(fruits[-1]) # cherry (last item) # Modifying lists fruits[1] = "blueberry" # Change item fruits.append("orange") # Add to end fruits.insert(0, "grape") # Insert at position fruits.remove("apple") # Remove by value last_fruit = fruits.pop() # Remove and return last item # List operations numbers = [1, 2, 3] print(len(numbers)) # 3 (length) print(2 in numbers) # True (membership test) print(numbers + [4, 5]) # [1, 2, 3, 4, 5] (concatenation) # List slicing numbers = [0, 1, 2, 3, 4, 5] print(numbers[1:4]) # [1, 2, 3] print(numbers[:3]) # [0, 1, 2] print(numbers[3:]) # [3, 4, 5] print(numbers[::2]) # [0, 2, 4] (every 2nd item) print(numbers[::-1]) # [5, 4, 3, 2, 1, 0] (reverse)

4. Tuples (Immutable Lists)

# Creating tuples coordinates = (10, 20) rgb = (255, 128, 0) single = (42,) # Note the comma for single-item tuple # Accessing elements (same as lists) print(coordinates[0]) # 10 # Tuples are immutable - can't change! # coordinates[0] = 15 # ERROR! # Tuple unpacking x, y = coordinates print(f"x={x}, y={y}") # x=10, y=20 # Use tuples for fixed collections person = ("Alice", 30, "Engineer") name, age, job = person

5. Dictionaries (Key-Value Pairs)

# Creating dictionaries person = { "name": "Alice", "age": 30, "city": "New York", "is_student": False } # Accessing values print(person["name"]) # Alice print(person.get("age")) # 30 print(person.get("job", "Unknown")) # Unknown (default) # Modifying dictionaries person["age"] = 31 # Update value person["job"] = "Engineer" # Add new key-value del person["is_student"] # Remove key # Dictionary methods print(person.keys()) # All keys print(person.values()) # All values print(person.items()) # All key-value pairs # Checking membership if "name" in person: print(f"Name: {person['name']}")

Control Flow

If Statements

# Basic if statement age = 18 if age >= 18: print("You are an adult") # If-else if age >= 18: print("You can vote") else: print("You cannot vote yet") # If-elif-else score = 85 if score >= 90: grade = "A" elif score >= 80: grade = "B" elif score >= 70: grade = "C" elif score >= 60: grade = "D" else: grade = "F" print(f"Grade: {grade}") # Comparison operators x, y = 10, 20 print(x == y) # False (equal to) print(x != y) # True (not equal to) print(x < y) # True (less than) print(x > y) # False (greater than) print(x <= y) # True (less than or equal) print(x >= y) # False (greater than or equal) # Logical operators age = 25 has_license = True if age >= 18 and has_license: print("Can drive") if age < 18 or not has_license: print("Cannot drive")

Loops

# For loop - iterate over sequences fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(fruit) # For loop with range for i in range(5): # 0 to 4 print(i) for i in range(1, 6): # 1 to 5 print(i) for i in range(0, 10, 2): # 0, 2, 4, 6, 8 print(i) # Enumerate - get index and value for index, fruit in enumerate(fruits): print(f"{index}: {fruit}") # While loop count = 0 while count < 5: print(count) count += 1 # Increment # Break and continue for i in range(10): if i == 3: continue # Skip 3 if i == 7: break # Stop at 7 print(i)

Functions

# Defining a function def greet(name): print(f"Hello, {name}!") # Calling the function greet("Alice") # Hello, Alice! # Function with return value def add(a, b): return a + b result = add(5, 3) print(result) # 8 # Default parameters def greet(name, greeting="Hello"): return f"{greeting}, {name}!" print(greet("Bob")) # Hello, Bob! print(greet("Bob", "Hi")) # Hi, Bob! # Multiple return values def get_user_info(): name = "Alice" age = 30 city = "NYC" return name, age, city name, age, city = get_user_info() # Variable-length arguments def sum_all(*numbers): total = 0 for num in numbers: total += num return total print(sum_all(1, 2, 3)) # 6 print(sum_all(1, 2, 3, 4, 5)) # 15

Practical Examples

Example 1: Temperature Converter

def celsius_to_fahrenheit(celsius): return (celsius * 9/5) + 32 def fahrenheit_to_celsius(fahrenheit): return (fahrenheit - 32) * 5/9 # Usage temp_c = 25 temp_f = celsius_to_fahrenheit(temp_c) print(f"{temp_c}¬∞C = {temp_f}¬∞F") temp_f = 77 temp_c = fahrenheit_to_celsius(temp_f) print(f"{temp_f}¬∞F = {temp_c:.1f}¬∞C")

Example 2: Simple Calculator

def calculator(num1, num2, operation): if operation == "+": return num1 + num2 elif operation == "-": return num1 - num2 elif operation == "*": return num1 * num2 elif operation == "/": if num2 != 0: return num1 / num2 else: return "Error: Division by zero" else: return "Invalid operation" # Usage print(calculator(10, 5, "+")) # 15 print(calculator(10, 5, "*")) # 50 print(calculator(10, 0, "/")) # Error: Division by zero

Example 3: Grade Analyzer

def analyze_grades(grades): if not grades: return "No grades provided" average = sum(grades) / len(grades) highest = max(grades) lowest = min(grades) passing = [g for g in grades if g >= 60] return { "average": round(average, 2), "highest": highest, "lowest": lowest, "passing_count": len(passing), "total_count": len(grades) } # Usage student_grades = [85, 92, 78, 90, 88] stats = analyze_grades(student_grades) print(f"Average: {stats['average']}") print(f"Highest: {stats['highest']}") print(f"Lowest: {stats['lowest']}") print(f"Passing: {stats['passing_count']}/{stats['total_count']}")

Summary

  • Python is beginner-friendly with clean, readable syntax
  • Variables don't need type declarations
  • Rich built-in data types: lists, tuples, dictionaries
  • Control flow with if/elif/else and for/while loops
  • Functions make code reusable and organized
  • String formatting with f-strings is powerful and concise
