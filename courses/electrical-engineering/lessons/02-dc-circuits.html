Introduction to DC Circuits

Direct Current (DC) circuits are the foundation of electrical engineering. Unlike AC (alternating current), DC flows in one direction only. Batteries, solar panels, and electronic devices all operate on DC principles.

  • Series and parallel resistor circuits
  • Voltage and current dividers
  • Kirchhoff's laws for complex circuits
  • Mesh and nodal analysis techniques
  • Thevenin and Norton equivalent circuits
  • Maximum power transfer theorem

Fundamental Laws Review

Ohm's Law

The most fundamental relationship in electrical circuits:

V = I √ó R Where: V = Voltage (Volts, V) I = Current (Amperes, A) R = Resistance (Ohms, Œ©) Derived forms: I = V / R R = V / I

Power Calculations

P = V √ó I (Power = Voltage √ó Current) P = I¬≤ √ó R (Using Ohm's law: V = IR) P = V¬≤ / R (Using Ohm's law: I = V/R) Where: P = Power (Watts, W) Energy: W = P √ó t (Energy = Power √ó time) W measured in Joules (J) or Watt-hours (Wh)

Given: A 12V battery connected to a 4Œ© resistor Find: Current and power dissipated Solution: I = V / R = 12V / 4Œ© = 3A P = V √ó I = 12V √ó 3A = 36W Or using: P = V¬≤ / R = 12¬≤ / 4 = 144 / 4 = 36W ‚úì

Current Divider Rule

For finding current distribution in parallel circuits:

For two resistors in parallel: I‚ÇÅ = I_total √ó (R‚ÇÇ / (R‚ÇÅ + R‚ÇÇ)) I‚ÇÇ = I_total √ó (R‚ÇÅ / (R‚ÇÅ + R‚ÇÇ)) Note: Current is inversely proportional to resistance! Example: I_total = 10A, R‚ÇÅ = 20Œ©, R‚ÇÇ = 30Œ© I‚ÇÅ = 10A √ó (30/(20+30)) = 10A √ó 30/50 = 6A I‚ÇÇ = 10A √ó (20/(20+30)) = 10A √ó 20/50 = 4A Check: 6A + 4A = 10A ‚úì

‚ö†Ô∏è Important Notes on Parallel Circuits

  • Total resistance is always less than the smallest individual resistance
  • Adding more parallel resistors decreases total resistance
  • More current flows through smaller resistances (path of least resistance)
  • Short circuits (R=0) make R_total = 0, causing infinite current (dangerous!)

Kirchhoff's Laws

These fundamental laws apply to ALL circuits and are essential for analyzing complex networks.

Kirchhoff's Current Law (KCL)

The algebraic sum of currents at any node equals zero.

KCL: Œ£ I_in = Œ£ I_out Or equivalently: Œ£ I_node = 0 (considering direction: in=+, out=-) Example at Node A: 2A ‚îÄ‚Üí ‚óè ‚Üê‚îÄ 3A ‚îÇ ‚Üì 5A KCL: 2A + 3A - 5A = 0 ‚úì Or: Currents in (5A) = Currents out (2A + 3A) ‚úì

Kirchhoff's Voltage Law (KVL)

The algebraic sum of voltages around any closed loop equals zero.

KVL: Œ£ V_loop = 0 Convention: - Voltage rises (- to +): positive - Voltage drops (+ to -): negative Example Loop: +12V battery, -4V resistor, -8V resistor KVL: +12V - 4V - 8V = 0 ‚úì

Mesh Analysis (Loop Current Method)

Systematic technique for analyzing circuits with multiple loops using KVL.

Mesh Analysis Steps:

  1. Identify all independent meshes (loops)
  2. Assign mesh currents (usually clockwise)
  3. Apply KVL to each mesh
  4. Solve simultaneous equations
  5. Find branch currents from mesh currents

Example 5: Two-Mesh Circuit

Given: Mesh 1: 12V source, R‚ÇÅ=4Œ©, R‚ÇÇ=8Œ© (shared) Mesh 2: R‚ÇÇ=8Œ© (shared), R‚ÇÉ=6Œ©, 6V source Find: Mesh currents I‚ÇÅ and I‚ÇÇ Solution: Mesh 1 (KVL): -12 + 4I‚ÇÅ + 8(I‚ÇÅ - I‚ÇÇ) = 0 -12 + 4I‚ÇÅ + 8I‚ÇÅ - 8I‚ÇÇ = 0 12I‚ÇÅ - 8I‚ÇÇ = 12 ... equation (1) Mesh 2 (KVL): 8(I‚ÇÇ - I‚ÇÅ) + 6I‚ÇÇ - 6 = 0 8I‚ÇÇ - 8I‚ÇÅ + 6I‚ÇÇ - 6 = 0 -8I‚ÇÅ + 14I‚ÇÇ = 6 ... equation (2) Solve simultaneously: From (1): I‚ÇÅ = (12 + 8I‚ÇÇ)/12 = 1 + 2I‚ÇÇ/3 Substitute into (2): -8(1 + 2I‚ÇÇ/3) + 14I‚ÇÇ = 6 -8 - 16I‚ÇÇ/3 + 14I‚ÇÇ = 6 14I‚ÇÇ - 16I‚ÇÇ/3 = 14 (42I‚ÇÇ - 16I‚ÇÇ)/3 = 14 26I‚ÇÇ/3 = 14 I‚ÇÇ = 42/26 = 1.615A I‚ÇÅ = 1 + 2(1.615)/3 = 2.077A

Nodal Analysis (Node Voltage Method)

Alternative systematic technique using KCL at nodes to find node voltages.

Nodal Analysis Steps:

  1. Select a reference node (ground, 0V)
  2. Assign voltage variables to other nodes
  3. Apply KCL at each non-reference node
  4. Express currents using Ohm's law: I = (V‚ÇÅ - V‚ÇÇ)/R
  5. Solve simultaneous equations for node voltages

Thevenin's Theorem

Any linear circuit with two terminals can be replaced by an equivalent voltage source (V_TH) in series with a resistance (R_TH).

Finding Thevenin Equivalent:

  1. V_TH: Open-circuit voltage at terminals
  2. R_TH: Resistance seen from terminals with sources deactivated
    • Voltage sources ‚Üí short circuit (0V)
    • Current sources ‚Üí open circuit (0A)

Example 7: Thevenin Equivalent

Given: 12V source in series with R‚ÇÅ=4Œ©, then parallel with R‚ÇÇ=12Œ© Terminals across R‚ÇÇ Find: V_TH and R_TH Solution: Step 1: V_TH (open circuit voltage) With R‚ÇÇ removed, voltage divider: V_TH = 12V √ó (12Œ©)/(4Œ© + 12Œ©) = 12V √ó 0.75 = 9V Step 2: R_TH Short 12V source, looking from terminals: R_TH = R‚ÇÅ || R‚ÇÇ = (4 √ó 12)/(4 + 12) = 48/16 = 3Œ© Thevenin Equivalent: 9V source in series with 3Œ©

Norton's Theorem

Alternative to Thevenin: equivalent current source (I_N) in parallel with resistance (R_N).

Conversion between Thevenin and Norton: I_N = V_TH / R_TH R_N = R_TH V_TH = I_N √ó R_N Example from above: I_N = 9V / 3Œ© = 3A R_N = 3Œ© Norton Equivalent: 3A source in parallel with 3Œ©

Maximum Power Transfer Theorem

Maximum power is transferred to a load when load resistance equals source resistance.

For maximum power transfer: R_Load = R_Source (or R_TH) Maximum power: P_max = V_TH¬≤ / (4 √ó R_TH) Efficiency at max power: 50% Example: V_TH = 12V, R_TH = 4Œ© For max power: R_Load = 4Œ© P_max = 12¬≤ / (4 √ó 4) = 144/16 = 9W

Summary

  • Series circuits: same current, voltages add, resistances add
  • Parallel circuits: same voltage, currents add, conductances add
  • KCL: Current in = Current out at any node
  • KVL: Voltage rises = Voltage drops around any loop
  • Mesh analysis uses KVL for loop currents
  • Nodal analysis uses KCL for node voltages
  • Thevenin/Norton simplify complex circuits for analysis
  • Maximum power transfer when R_Load = R_Source
